从github导出备份的hexo源文件

安装node，git，hexo

将备份的hexo分支导出到本地

在导出的文件夹下面，安装需要的依赖

注意：

默认首次安装git，需要初始化邮箱和用户名

hexo首次部署需要生成公钥和私钥

使用ssh-keygen生成私钥和公钥

登录Github，点击头像下的settings，在左侧店家SSH and GPG keys

添加ssh,新建一个new ssh key，将id_rsa.pub文件里的内容复制上去。

测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了。
